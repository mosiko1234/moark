Problem: "command not found: moark-pack"
Solution: Make sure pip's bin directory is in your PATH.
          Try: python3 -m moark_pack.pack --help

Problem: "SSL certificate verification failed"
Solution: Use --insecure flag or check "Disable SSL Verification" in UI.

Problem: "HTTP Basic: Access denied"
Solution: Your GitLab instance might require a Personal Access Token.
          Create one at: GitLab Settings â†’ Access Tokens
          Use the token as the password instead of your regular password.

Problem: "Got unexpected extra argument"
Solution: Make sure you're running the command in a single line.
          Avoid using backslashes (\) for line continuation in xonsh shell.
          Instead, run the entire command on one line.

Problem: UI doesn't start
Solution: Make sure textual is installed: pip install textual>=0.40.0
